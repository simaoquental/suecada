Projeto desenvolvido para a cadeira de Desenvolvimento para Web do curso de Informatica de Gestão do ISCAC

Passos para correr o programa:
1 - Fazer o download do Repositorio
2 - Entrar no diretorio "servidor"
3 - correr o comando "npm install"
4 - ligar o servidor "node index.js"
5 - Abrir no browser "localhost:3000"
